# IP Address

## What is an IP Address?
An IP (Internet Protocol) Address is a unique address assigned to every device connected to a network. It helps devices identify and communicate with each other.

## Types
- IPv4 (Example: 192.168.1.1)
- IPv6 (Example: 2001:db8::1)

## Types of IP
- Public IP
- Private IP
- Static IP
- Dynamic IP

## Summary
An IP address is used to identify devices and enable communication over a network.
